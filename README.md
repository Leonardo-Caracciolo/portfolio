# Portfolio — Leonardo Caracciolo

Portfolio web personal desarrollado con Astro + React + Tailwind CSS.

## 🚀 Stack
- **Framework:** Astro 4 + React
- **Estilos:** Tailwind CSS
- **Deploy:** Vercel
- **Dev environment:** GitHub Codespaces

## 📋 Secciones
- Hero con animación de tipado
- Sobre mí
- Experiencia (Deloitte · Freelance · PwC)
- Proyectos destacados
- Stack tecnológico
- Chat con IA (próximamente)
- Contacto

## 🛠️ Comandos

| Comando | Acción |
|---|---|
| `npm install` | Instala dependencias |
| `npm run dev` | Servidor local en `localhost:4321` |
| `npm run build` | Build de producción en `./dist/` |
| `npm run preview` | Preview del build local |

## 📦 Instalación

```bash
git clone https://github.com/Leonardo-Caracciolo/portfolio
cd portfolio
npm install
npm run dev
```
